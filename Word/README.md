# Microsoft Word 2021 Training – Day 02 & Day 03

This repository contains the exercises, practical tasks, and notes for **Microsoft Word 2021** training.

---

## Day 02: Microsoft Word 2021 – Interface & Formatting (Part 1)

### Topics Covered:
- **Word Interface Overview**  
  - Ribbon, Tabs, Status Bar
- **Creating, Saving, and Opening Documents**
- **Applying Formatting**  
  - Font, Alignment, Styles
- **Creating Lists and Tables**
- **Using Page Setup**  
  - Margins, Orientation

### Practical Task:
- **Create a professional letter with a table**  
  - Include header and footer  
  - Insert a table for product details or information  
  - Apply basic formatting (font style, alignment, bold/italics)

---

## Day 03: Microsoft Word 2021 – Templates, Forms, Mail Merge (Part 2)

### Topics Covered:
- **Inserting Images, Shapes, and Icons**
- **Working with Headers, Footers, and Watermarks**
- **Creating Templates and Forms**
- **Mail Merge with Excel Source**
- **Proofing Tools**  
  - Spelling, Grammar, Thesaurus
- **Integrating AI Tools in Word**  
  - Editor, Designer

### Practical Task:
- **Create a certificate template using Mail Merge**  
  - Insert placeholders for Name, Course, Date  
  - Add design elements: images, shapes, icons  
  - Use header, footer, and watermark  
  - Proofread using Spelling & Grammar  
  - Apply AI tools (Editor / Designer) if available  

---

## Notes:
- All practical tasks were completed successfully.  
- Documents were saved in proper format: `.docx` or `.dotx` for templates.  
- Mail Merge was demonstrated with a sample Excel sheet.  
- AI tools were used to improve writing and layout where available.
